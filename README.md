# Ashwake Armour & Weapons

Forged in ash and awakened by lost knowledge, Ashwake Armour & Weapons introduces powerful gear shaped by ancient magic and evolving technology, designed to define survival and progression in the Ashwake world.

## Status
In active development.

## Development
Requires JDK 21. Use the Gradle wrapper for common tasks:

- `gradlew runClient`
- `gradlew runServer`
- `gradlew build`
